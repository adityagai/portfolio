# Aditya Gaitonde — Portfolio

A personal profile website for **Aditya Gaitonde**, Senior Engineering Manager specializing in cloud infrastructure and distributed systems.

🔗 **Live site:** https://adityagai.github.io/portfolio

## About

A single-page, fully responsive portfolio built as one self-contained `index.html` — no build step, no dependencies to install. It presents a professional summary, career timeline, core competencies, education, and recognition.

## Tech

- Vanilla **HTML, CSS, and JavaScript** in a single file
- **Google Fonts** (Bricolage Grotesque, IBM Plex Sans, IBM Plex Mono)
- A lightweight **canvas node-network** animation in the hero (nods to distributed systems)
- Scroll-reveal animations via `IntersectionObserver`
- Dark theme, fully responsive, and respects `prefers-reduced-motion`

## Run locally

No tooling required — just open the file in a browser:

```bash
open index.html        # macOS
# or simply double-click index.html
```

## Deployment

Hosted with **GitHub Pages**:

1. Settings → Pages
2. Source: **Deploy from a branch** → Branch: `main` → folder: `/ (root)`
3. The site publishes at `https://adityagai.github.io/portfolio`

A custom domain (`adityagaitonde.com`) can be added later under Settings → Pages → Custom domain.
